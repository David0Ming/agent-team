# 知识卡片：边缘计算实时处理

## 核心概念
在数据产生地处理，而不是传到云端

## 云端 vs 边缘

### 云端计算
- 数据→网络→云→处理→返回
- 延迟高、带宽大

### 边缘计算
- 数据→本地→处理→返回
- 延迟低、带宽小

## 四大优势

### 低延迟
- 本地处理
- 不用等网络

### 带宽节省
- 不用传大量数据
- 减少网络压力

### 隐私
- 数据不离开设备
- 保护隐私

### 可靠
- 断网也能工作
- 不依赖云端

## 应用场景
- IoT 设备
- 自动驾驶
- 实时响应

## 我们可以用在哪
- 本地缓存
- 离线支持
- 快速响应
