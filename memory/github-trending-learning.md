# GitHub Trending 学习 - 2026-03-05

## 记录原则

每次学习必须包含:
1. **是什么** - 技术描述
2. **应用** - 我们系统哪里可以用？我们需要吗？我们已经掌握了吗？← 扩充
3. **时机** - 使用时机
4. **怎么用** - 调用方式
5. **验证** - 真正掌握了吗？能复现/应用吗？
