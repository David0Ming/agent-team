## 1️⃣ 复述

**Superpowers** 是一个 AI 编程代理的完整软件开发工作流系统。

**核心理念**: 让 AI 像有经验的工程师一样工作，而不是像"会写代码但不懂工程"的实习生。

**4个核心原则**:
1. **TDD** — 先写测试，没看到测试失败就不算
2. **系统化** — 用流程替代猜测
3. **简洁** — YAGNI 原则
4. **证据** — 必须验证，不能"我觉得"

**5阶段工作流**:
1. 头脑风暴 — 苏格拉底式提问，厘清需求
2. 工作区隔离 — 用 git worktree
3. 编写计划 — 拆解成 2-5 分钟的小任务
4. 子代理驱动 — 主代理派遣子代理，2阶段审查
5. 收尾 — 验证测试，合并/PR

**14个核心技能**:
- 测试类: test-driven-development
- 调试类: systematic-debugging, verification-before-completion
- 协作类: brainstorming, writing-plans, requesting/receiving-code-review

## 2️⃣ 应用

**我们可以用**:
1. ✅ 提升代码质量 — 用 TDD + 两阶段审查
2. ✅ 复杂任务拆分 — 用 writing-plans
3. ✅ 子代理并行 — 用 dispatching-parallel-agents
