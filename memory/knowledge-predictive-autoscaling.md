# 知识卡片：预测性自动伸缩

## 核心概念
根据预测自动调整资源

## vs 传统自动伸缩
- 传统：负载高 → 才扩容 → 有延迟
- 预测性：预测负载高 → 提前扩容 → 零延迟

## 核心阶段

### 预测
- ML 预测未来负载
- 时间序列分析

### 规划
- 计算需要多少资源
- 规划扩容方案

### 执行
- 提前扩容/缩容
- 无感知调整

## 算法
- 时间序列预测
- 趋势分析
- 周期性识别

## 我们可以用在哪
- 预测 API 负载 → 提前准备
- 预测模型需求 → 提前切换
- 预测资源需求 → 提前分配
