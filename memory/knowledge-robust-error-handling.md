# 知识卡片：健壮错误处理

## 核心概念
系统优雅处理各类错误的机制

## 核心要素

### 错误分类
- 区分错误类型
- 可恢复/不可恢复

### 错误恢复
- 自动恢复策略
- 重试/降级/回滚

### 错误传播
- 正确传递错误
- 不吞掉错误

### 错误日志
- 记录错误上下文
- 便于调试

## 最佳实践
- 不要忽略错误
- 给用户友好提示
- 记录错误上下文
- 优雅降级

## 我们可以用在哪
- API 调用错误处理
- 模型失败恢复
- 用户提示优化
