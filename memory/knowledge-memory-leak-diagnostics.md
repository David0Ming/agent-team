# 知识卡片：AI 内存泄漏诊断

## 核心概念
用 AI 自动检测和定位内存泄漏

## 传统 vs AI
- 传统：人工分析、经验猜测、耗时
- AI：自动分析、识别模式、快速定位

## 常见泄漏模式

### 缓存泄漏
- 缓存无限增长
- 未设置上限

### 事件监听器
- 未移除的监听器
- 累积

### 闭包引用
- 闭包持有引用
- 无法GC

### 全局变量
- 全局变量累积
- 永生

## 我们可以用在哪
- 检测内存泄漏
- 预防 OOM
- 优化内存使用
