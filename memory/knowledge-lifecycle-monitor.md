# 知识卡片：强化生命周期监控 (Hardened Lifecycle Monitor)

## 核心概念
让监控系统出错时也能优雅活下去

## 关键能力

### 1. 错误捕获
- 捕获 JSON 解析错误
- 捕获进程异常

### 2. 优雅降级
- 正常检测失败 → 降级到启发式检测
- 不因为一次失误而崩溃

### 3. 防止死循环
- 防止 watchdog_crash 无限循环
- 超时保护

## 应用场景
- Gateway 进程监控
- API 失败自动恢复
- 心跳检测降级

## 我们可以用在哪
- API 冷却时的优雅处理
- 监控脚本防崩溃
- 自动重试机制
