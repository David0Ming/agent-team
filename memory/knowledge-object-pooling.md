# 知识卡片：对象池内存优化

## 核心概念
重用对象而不是频繁创建销毁

## 核心问题
- 创建对象：耗时
- 销毁对象：耗时
- 垃圾回收：耗时

## 解决方案

### 对象池
- 预创建对象
- 用完归还复用

### 连接池
- 数据库连接复用
- 减少连接开销

### 线程池
- 线程复用
- 避免频繁创建

## 应用场景
- API 请求处理
- 数据库连接
- 并发任务

## 我们可以用在哪
- HTTP 连接复用
- 数据库连接池
- 任务队列
