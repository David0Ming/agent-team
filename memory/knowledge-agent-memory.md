# 知识卡片：Agent 会话记忆管理

## 核心概念
AI Agent 跨会话记住之前上下文的能力

## 核心挑战
1. 会话结束 → 记忆丢失
2. 长对话 → 上下文太长
3. 多会话 → 知识不共享

## 三个记忆层次

### 短期记忆
- 会话内上下文
- 随会话结束清除

### 中期记忆
- 会话间临时存储
- 持久化到下次会话

### 长期记忆
- 跨会话持久化
- 选择性记忆 + 遗忘

## 关键技术
- 记忆摘要：压缩关键信息
- 选择性记忆：记住重要的
- 遗忘机制：忘记不重要的
- 检索增强：记得住、找得到

## 我们可以用在哪
- API 中断后会话恢复
- 上下文智能压缩
- 长对话管理
