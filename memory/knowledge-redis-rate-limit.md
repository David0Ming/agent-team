# 知识卡片：Redis Lua 滑动窗口限流

## 核心概念
用 Redis + Lua 实现高性能限流

## 为什么用 Lua
- 原子性：脚本不被中断
- 性能：减少网络往返
- 灵活：可编程逻辑

## 限流算法

### 计数器
- 简单但不平滑
- 固定时间窗口

### 滑动窗口
- 平滑精确
- 任意时刻精确控制

### 令牌桶
- 允许突发流量
- 匀速消费

## 应用场景
- API 请求限流
- 防止过载
- 保护后端

## 我们可以用在哪
- API 限流
- 防止触发冷却
- 流量控制
