# 知识卡片：记忆化与懒加载

## 记忆化 (Memoization)
记住计算结果，避免重复计算

### 核心思想
- 输入相同 → 输出缓存
- 比喻：背乘法表

### 应用
- 递归结果缓存
- 重复计算优化

## 懒加载 (Lazy Loading)
延迟加载，用到时才加载

### 核心思想
- 延迟初始化
- 比喻：按需拿快递

### 应用
- 大数据分页
- 远程资源加载

## 我们可以用在哪
- 配置读取缓存
- 大批量日志分页
- API 响应缓存
