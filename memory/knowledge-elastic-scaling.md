# 知识卡片：云计算弹性扩展

## 核心概念
根据负载自动调整计算资源

## 扩展类型

### 水平扩展 (Horizontal)
- 增加机器数量
- 加服务器

### 垂直扩展 (Vertical)
- 增加单机配置
- 加CPU/内存

### 自动伸缩 (Auto Scaling)
- 根据负载自动扩缩
- 设置阈值触发

## 云原生特性
- 按需付费
- 弹性伸缩
- 高可用

## 应用场景
- API 流量高峰
- 突发流量响应
- 成本优化

## 我们可以用在哪
- API 负载均衡
- 流量高峰扩容
- 成本控制
