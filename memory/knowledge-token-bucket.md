# 知识卡片：令牌桶算法

## 核心概念
用令牌控制请求通过的限流算法

## 工作原理
- 桶里放令牌
- 每秒放入 N 个
- 每个请求消耗 1 个
- 没令牌 → 拒绝/等待

## vs 滑动窗口

### 滑动窗口
- 固定时间窗口
- 不允许突发

### 令牌桶
- 允许短暂突发
- 更平滑

## 优势
- 允许突发流量
- 平滑限流
- 实现简单

## 应用场景
- API 限流
- 流量控制
- 突发处理

## 我们可以用在哪
- API 限流
- 流量平滑
- 突发处理
