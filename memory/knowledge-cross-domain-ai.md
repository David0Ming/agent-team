# 知识卡片：跨领域 AI

## 核心概念
能处理多种类型任务/领域的 AI

## 单领域 vs 跨领域

### 单领域 AI
- 只能做翻译
- 只能做代码
- 只能做对话

### 跨领域 AI
- 一个模型，多种能力
- 代码+对话+推理+规划

## 核心挑战

### 领域知识冲突
- 不同领域知识可能矛盾

### 能力平衡
- 某种能力太强，其他变弱

### 上下文管理
- 多领域上下文切换

## 应用场景
- 多功能 Agent
- 综合助手
- 复杂任务

## 我们可以用在哪
- 多功能 Agent
- 灵活任务切换
- 综合能力
