## 🎯 优秀Prompt共同特点

### 1. 故事结构（三幕式）

```
第一幕（0-3秒）：建立冲突/悬念
  ↓
第二幕（3-9秒）：揭示/反转/互动  
  ↓
第三幕（9-12秒）：收尾/情感升华
```

### 2. 情绪曲线

```
案例1：恐惧 → 惊讶 → 温暖 → 俏皮
案例2：焦虑 → 惊喜 → 安心 → 豁然
案例3：困惑 → 迷失 → 发现 → 满足
案例4：惊奇 → 错位 → 细腻 → 和谐
```

### 3. 镜头语言丰富

| 手法 | 效果 | 案例 |
|------|------|------|
| 超低角度仰拍 | 压迫感/巨物感 | 案例1 |
| Dolly Zoom | 揭示/反转 | 案例1 |
| 360度环绕 | 全面展示品牌 | 案例1,3 |
| 手持抖动 | 焦虑/迷失感 | 案例2 |
| Tilt-shift | 微缩奇观 | 案例4 |
| 微距拍摄 | 细节质感 | 案例3 |

### 4. 品牌展示技巧

- **大字占比**：画面50%-80%
角度**：侧- **多种光、逆光、顺光、雨天反光
- **细节特写**：划痕、灰尘、水珠
- **对比手法**：巨物vs微小、迷雾vs清晰

### 5. 音效/节奏配合

- 怪兽低频震动 → 突然刹车（可爱版）
- 急速旋转 → 突然穿透
- 视觉反转 → 音效配合

## 💡 泽钢核心经验

1. **反差感**：巨物vs微小、恐怖vs温情
2. **锚点感**：大字作为视觉中心/安全符号
3. **细节感**：划痕、水珠、光影
4. **互动感**：司机角色+乘客视角
5. **节奏感**：急速vs舒缓、紧张vs放松
