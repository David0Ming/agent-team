## 💡 核心经验（泽钢总结）

1. **视觉反差**：黑白vs彩色、冷vs热、影子vs真实
2. **空间穿越**：穿过文字、车内漫游
3. **时间魔法**：倒放、重组、碎片拼接
4. **材质转换**：积木→真实、乐高质感
5. **科技滤镜**：热成像、红外视觉
6. **情感人设**：司机互动、手势、表情
