# 嘟嘟巴士Prompt - 司机篇（5条）

## 🎬 Prompt 1：司机的早餐

```
我要一个[生活气息]开头的12秒温情短片。

【0-3秒｜前提】：清晨6点，街道空旷。一个中年司机在路边摊吃早餐，手里拿着豆浆。

【3秒转场】：镜头从早餐摊摇到路边——一辆红色巴士驶来。司机放下豆浆，走向巴士。

【7秒内容】：
- 司机整理制服，登上驾驶座
- 乘客陆续上车，有学生、白领、老人
- 司机和每个乘客点头打招呼
- 车厢温暖，朝阳透过车窗洒进来

【结尾】：司机发动引擎，红色巴士驶入晨光中。字幕："每一次出发，都是新的故事"
```

## 🎬 Prompt 2：司机的秘密

```
我要一个[反转惊喜]开头的12秒治愈短片。

【0-3秒｜悬念】：晚高峰，车流拥堵。乘客们表情焦虑，不断看手机。

【3秒转场】：司机突然按下广播按钮，温柔的音乐响起。乘客们抬头。

【7秒内容】：
- 司机跟着音乐轻轻点头
- 镜头扫过车厢，乘客们放松下来，有人微笑
- 一位小朋友跟着节奏轻轻晃动手臂
- 司机从后视镜看到，微微一笑

【结尾】：交通疏通，巴士平稳启动。字幕："有些秘密，是把烦恼变成歌"
```

## 🎬 Prompt 3：司机的约定

```
我要一个[承诺坚守]开头的12秒情感短片。

【0-3秒｜回忆】：泛黄照片——年轻的司机和一个小女孩在巴士旁合影。

【3秒转场】：照片变成现实，同样的巴士，只是司机老了。小女孩背着书包跑来。

【7秒内容】：
- 小女孩跳上巴士，像小时候一样
- 司机看到她，笑着说："你长大了"
- 小女孩坐下窗边位置，像多年前一样
- 窗外风景流动，时光交错

【结尾】：巴士驶过城市。字幕："每一次终点，都是新的起点"
```

## 🎬 Prompt 4：司机的眼睛

```
我要一个[观察者视角]开头的12秒人文短片。

【0-3秒｜城市速写】：城市街景，行人匆匆。镜头从窗外扫过，像一双观察的眼睛。

【3秒转场】：镜头定格在驾驶座——司机专注的眼神，通过后视镜看着乘客。

【7秒内容】：
- 看到带孩子的妈妈，司机帮忙抬婴儿车
- 看到老人上车，司机多等一等
- 看到学生睡着，空调温度调高一点
- 看到情侣吵架，默默放轻音乐

【结尾】：后视镜里，司机微微笑。字幕："这座城市的故事，他都看过"
```

## 🎬 Prompt 5：司机的魔法

```
我要一个[奇幻治愈]开头的12秒创意短片。

【0-3秒｜魔幻时刻】：雨夜，街道反射着霓虹。巴士行驶在路上，车灯像魔法棒。

【3秒转场】：司机挥动方向盘，车灯扫过——奇迹发生！

【7秒内容】：
- 车灯扫过，积水变成彩虹
- 车灯扫过，乘客的表情从疲惫变成惊喜
- 车灯扫过，雨滴变成星星
- 车厢内，暖黄色的灯像一个小太阳

【结尾】：雨停，巴士驶入彩虹尽头。字幕："平凡的路，因为守护而发光"
```
