# 2026-02-28 - Token Estimator Skill 学习记录

## 今日成果

### 1. 学习了 OpenClaw Skill 系统
- **Skill vs Plugin**: 
  - Skill = 指导 Agent 的文档 + 脚本（放在 `skills/`）
  - Plugin = 运行时扩展（放在 `plugins/`）
- **Skill 结构**: SKILL.md (frontmatter + body) + scripts/ + references/ + assets/
- **渐进式披露**: 元数据 → SKILL.md → 引用文件（按需加载）

### 2. 创建了 token-estimator Skill
**功能**: 自动估算代码任务 token 成本，推荐最优模型

**阈值规则**（基于泽钢的需求）：
| 总字符数 | 估算 Token | 推荐模型 |
|---------|-----------|---------|
| < 3M | < 750K | Lightweight (Opus 4.6 / GPT-Mini) |
| 3M - 12M | 750K - 3M | Standard (GPT-5.1 / Kimi) |
| > 12M | > 3M | Premium (GPT-5.2 / Claude Opus) |

**文件位置**:
- Skill: `~/.openclaw/workspace/skills/token-estimator/`
- 打包: `~/.openclaw/workspace/token-estimator.skill`

**使用方法**:
```bash
# 估算当前目录
python ~/.openclaw/workspace/skills/token-estimator/scripts/estimate.py --path ./src --recommend

# 应用到配置
openclaw config set agent.model.primary "fox/gpt-5.2"
```

### 3. 学习的 Skill 设计原则
- **Concise is Key**: 上下文窗口是公共资源，只添加 Codex 不知道的信息
- **自由度匹配**: 脆弱操作用脚本（低自由度），灵活任务用文字指导（高自由度）
- **渐进式披露**: 大文件放 references/，SKILL.md 保持精简
