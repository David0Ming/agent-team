## 📝 学习原则

### 1. 深度学习5步法
1. **复述**: 用自己的话解释这个知识
2. **应用**: 思考哪里可以用这个知识
3. **时机**: 明确什么时候使用
4. **验证**: 验证是否真正掌握
5. **记忆**: 存入知识卡片
