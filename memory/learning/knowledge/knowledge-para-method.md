# 知识管理最佳实践 - PARA方法应用

## PARA方法核心

**4个分类**：
1. **Projects（项目）**：短期目标，有明确完成标准
2. **Areas（领域）**：持续责任，无结束日期  
3. **Resources（资源）**：学习材料和感兴趣的主题
4. **Archives（归档）**：不再活跃的内容

**核心原则**：
- 按可操作性组织（Organize by actionability）
- 项目驱动（围绕具体目标，不是学科）
- 简单到极致（减少维护成本）
