## 2026-03-20

**多Agent + 统一人工审核，比单Agent顺序流更稳**
- 是什么：2026年agentic coding趋势强调：单Agent顺序处理受限于单一上下文，多Agent编排把需求、架构、实现、验证拆到专用上下文并行处理，最后由人统一审核
- 为什么有价值：能同时提升速度、上下文清晰度和质量控制，尤其适合中大型开发任务
- 来源：Anthropic 2026 Agentic Coding Trends Report / McKinsey QuantumBlack agentic workflows

**评估AI Agent时，可靠性比能力堆高更重要**
- 是什么：近期讨论强调，agent系统的真实价值不只看能力上限，更看在开放环境下是否稳定、可控、可评估
- 为什么有价值：做AI应用落地时，优先优化可靠性、边界和评测体系，比盲目追求更强agent更实用
- 来源：Hacker News discussion / NIST AI Agent Security public comment 2026
