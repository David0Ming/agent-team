# 知识卡片：分布式追踪 (Distributed Tracing)

## 核心概念
追踪请求在多个服务间的完整路径

## 核心术语

### Trace
- 完整请求路径
- 从请求到响应的全过程

### Span
- 单个服务的处理单元
- 包含：开始时间、结束时间、名称

### Parent
- 父级 Span
- 形成调用链路

## 应用场景
- 定位请求慢的原因
- 找出哪个服务出错
- 分析复杂调用链路

## 我们可以用在哪
- API 耗时分析
- 错误根因定位
- 请求链路可视化
