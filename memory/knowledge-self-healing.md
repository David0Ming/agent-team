# 知识卡片：自愈 AI (Self-healing AI)

## 核心概念
能够自动检测和修复问题的 AI 系统

## 核心能力

### 异常检测
- 自动发现问题
- 监控异常模式

### 根因分析
- 找出问题原因
- 不只是症状

### 自动修复
- 尝试修复问题
- 不需要人介入

### 学习预防
- 从错误中学习
- 防止再次发生

## 与传统系统区别
- 传统：出问题 → 人来修
- 自愈：出问题 → AI 自动修

## 我们可以用在哪
- API 失败自动切换
- 异常行为自动检测
- 错误模式学习预防
