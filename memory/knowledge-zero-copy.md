# 知识卡片：零拷贝 (Zero-copy)

## 核心概念
数据在内存之间传输时不经过 CPU

## 传统拷贝
```
磁盘 → 内核缓冲区 → 用户空间 → CPU → 网络缓冲区 → 网络
```

## 零拷贝
```
磁盘 → 内核缓冲区 → 网络缓冲区 → 网络
```

## 核心优势
- 减少 CPU 参与
- 减少内存复制
- 提升性能

## 应用场景
- 大文件传输
- 日志处理
- API 响应

## 我们可以用在哪
- 大数据传输
- 性能优化
