# 知识卡片：日志聚合管道 (Logging Pipeline)

## 核心概念
分散日志自动收集、处理、存储的系统

## 三个核心步骤
| 步骤 | 功能 |
|------|------|
| 收集 | 从各服务抓取日志 |
| 处理 | 过滤、格式化、提取关键信息 |
| 存储 | 高效存储，支持快速查询 |

## 上层应用
- 实时监控
- 异常报警
- 可视化 Dashboard
- 趋势分析

## 应用场景
- 所有 Agent 操作日志自动收集
- API 失败次数统计 → 触发熔断
- 异常模式识别 → 自动报警

## 我们可以用在哪
- 心跳日志持久化存储
- API 失败率监控
- Agent 行为分析
