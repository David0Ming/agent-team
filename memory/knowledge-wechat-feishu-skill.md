## 1️⃣ 复述

**核心功能**: 用 OpenClaw (小龙虾) 的 skill 机制，实现「收到微信文章链接 → 自动总结 + 打标签 → 同步到飞书表格」

**实现步骤**:
1. 用 `AskUserQuestion` 工具采访需求，生成 SPEC.md
2. 把 SPEC.md 丢给 Claude Code 生成 SKILL.md
3. 配置飞书4个凭证（APP_ID, APP_SECRET, SPREADSHEET_TOKEN, SHEET_ID）
4. 把 SKILL.md 放到小龙虾 skills 目录

**Skill 核心流程**:
1. 抓取页面内容
2. 提取结构化信息（标题、摘要、关键词、分类）
3. 获取飞书 token
4. 写入飞书表格

---

## 2️⃣ 应用

**我们可以用在**:
1. ✅ 收藏夹管理：自动整理看到的好文章
2. ✅ 信息采集：批量抓取 + 分类
3. ✅ 飞书集成：做自己的知识库
