# 知识卡片：预测性监控

## 核心概念
在问题发生前预测并预警

## vs 传统监控
- 传统：问题发生 → 报警
- 预测性：问题发生前 → 预警

## 核心技术

### 趋势分析
- 分析指标趋势
- 预测走势

### 异常检测
- 发现异常模式
- 提前发现

### 机器学习
- 预测未来
- 更准确

### 阈值预测
- 预测何时达阈值
- 提前准备

## 监控指标
- CPU 使用率趋势
- 内存增长趋势
- 错误率趋势

## 我们可以用在哪
- API 失败预测 → 提前切换
- 内存增长预警 → 提前清理
- 负载预测 → 提前扩容
