# 知识卡片：AI Agent 调试

## 核心概念
调试 AI Agent 的特殊技术

## 调试挑战
- 传统调试：断点+单步
- AI Agent：思维不可见
- 隐式决策：为什么这么回答？

## 调试技术

### 思维链 (Chain of Thought)
- 显示推理过程
- 方便定位问题

### 中间输出
- 检查每一步结果
- 找出错的环节

### 轨迹追踪
- 记录完整执行路径
- 回溯问题根源

### 对比测试
- 改变输入看输出
- 找出关键因素

## 调试工具
- LangSmith
- OpenAI Playground
- AgentOps

## 我们可以用在哪
- Agent 行为调试
- Prompt 优化
- 决策过程分析
