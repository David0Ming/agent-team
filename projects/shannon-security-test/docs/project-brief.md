# Shannon安全测试项目 - 讨论背景

## 项目目标
使用Shannon AI渗透测试工具，在本周某个时间测试嘟嘟巴士主营产品的安全性。

## 工具信息
- **Shannon**：全自动AI渗透测试工具（白盒）
- **GitHub**：https://github.com/KeygraphHQ/shannon
- **功能**：识别SQL注入、XSS、SSRF、认证/授权漏洞
- **特点**：主动执行攻击验证，"No exploit no report"原则
- **要求**：需要源代码、Docker、Anthropic API key
- **成本**：约$50/次，耗时1-1.5小时

## 关键约束
1. **测试时间**：低峰期（中午11:00-下午17:00）
2. **授权**：已获得主管授权
3. **环境**：⚠️ 不能在生产环境运行（只能staging/dev）
4. **风险**：破坏性测试，可能修改/删除数据
