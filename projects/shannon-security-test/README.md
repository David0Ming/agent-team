使用Shannon AI渗透测试工具对嘟嘟巴士主营产品进行安全性评估。

## 关键信息

### 工具特性
- **Shannon**：全自动AI渗透测试工具（白盒）
- **GitHub**：https://github.com/KeygraphHQ/shannon
- **功能**：SQL注入、XSS、SSRF、认证/授权漏洞检测
- **要求**：源代码、Docker、Anthropic API key
- **成本**：约$50/次，耗时1-1.5小时

### 约束条件
- 测试时间：低峰期（11:00-17:00）
- 环境限制：仅staging/dev，禁止生产环境
- 授权状态：已确认
