# 嘟嘟巴士先锋引擎 - 项目规范

## 🎯 项目原则（必须遵守）

> **核心原则**：所有生成的视频Prompt必须融入真实的嘟嘟巴士信息，用于品牌宣传

### 必须包含的信息元素

1. **价格优势**
   - 最低5.8元起（东莞↔广州）
   - 比打车便宜90%+
   
2. **覆盖范围**
   - 21个城市
   - 800条线路
   - 广东大湾区
   
3. **便捷服务**
   - 微信扫码购票
   - 7:00-22:00服务时间
   - 电子票凭证

4. **热门线路**
   - 东莞↔广州：5.8元起
   - 佛山↔广州：10元起
   - 珠海↔广州：19.9元起
   - 深圳↔广州：25.8元起
   - 广州↔中山：最快0.7小时

5. **品牌展示**
   - 车身印有'嘟嘟巴士'四个大字的红色巴士

### 视频内容要求

- ✅ 必须融入真实线路信息（价格/时长/站点）
- ✅ 必须体现价格优势
- ✅ 必须展示便捷服务特点
- ✅ 必须包含品牌展示（车身大字）

## 定时任务
- **每日执行时间**：09:00

## 竞品数据要求

### 必须获取的数据字段
- 粉丝数
- **播放量**
- **点赞量**
- 评论数
- 转发数
- 作品数量
- 发布时间
- **视频内容**（主题、风格、形式）

### 数据获取规则
- **只获取昨日数据**
- 计算方式：今日累计 - 昨日累计 = 昨日单日数据
- 如果只能获取近7日数据，则用今日-昨日=昨日

### 数据分析
- 视频内容分析（主题、风格、形式）
- 热门视频规律
- 用户互动特点

### 模仿方案
- 可复制的视频形式
- 内容选题建议
- 差异化竞争点

## 视频规格

### 基本规格
- 时长：≤12秒
- 画幅：9:16 竖屏

### 品牌展示（强制替换规则）
> **提示词生成时，所有"巴士"相关描述必须替换为：**
> "车身印有'嘟嘟巴士'四个大字的红色巴士"

## 内容要求
- 禁止出现其他巴士logo、水印
- **必须出现**：车身印有'嘟嘟巴士'四个大字的红色巴士

### 司机人设
- 年龄：42岁
- 性格：爱吐槽但暖心
- 口头禅：老铁们 / 家人们 / 这趟车稳了
- 语言风格：广深普通话特色

## Prompt生成流程

1. **热点搜索** → DMing负责（昨日热点）
2. **竞品数据** → DMing负责（昨日数据）
3. **Prompt生成** → David负责
   - 结合热点
   - 融入人设
   - Seedance格式
   - **强制品牌描述替换**
4. **格式检查** → 自动检查
5. **Tag建议** → 动态生成

## Tag规范
- 固定Tag：#嘟嘟巴士
- 动态Tag：根据热点内容决定
