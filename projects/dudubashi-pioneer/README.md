# 嘟嘟巴士先锋引擎 - 项目规范

## 视频要求

### 基本规格
- 时长：≤12秒
- 画幅：9:16 竖屏
- 品牌展示：**车身必须有"嘟嘟巴士"大字**

### 内容要求
- 前3秒必须有冲突或笑点（Hook）
- 每3秒一切换镜头
- 结尾必须有反转 + 手势"这趟车稳了"
- 禁止出现其他巴士logo、水印

### 司机人设
- 年龄：42岁
- 性格：爱吐槽但暖心
- 口头禅：老铁们 / 家人们 / 这趟车稳了
- 语言风格：广深普通话特色

## Prompt生成流程

1. **热点搜索** → DMing负责

5. **Tag建议** → 动态生成
