# 隐私保护框架

## 当前数据处理现状

### 抓取的数据类型
- 竞品公开内容（小红书、抖音、视频号）
- 热点话题、关键词
- 生成的视频Prompt

### 潜在隐私风险
1. **第三方平台数据**: 抓取的竞品内容可能包含用户生成内容(UGC)
2. **存储安全**: 抓取的数据如何存储
3. **数据保留**: 数据保留多长时间

## 隐私保护原则

### 1. 数据最小化
- 只抓取必要的公开信息
- 不抓取个人隐私数据（联系方式、身份证等）

### 2. 目的限制
- 抓取数据仅用于：竞品分析、内容创意
- 不得用于：商业销售、用户画像

### 3. 存储安全
- 敏感数据加密存储
- **数据保留期限：30天**
- **删除前必须询问泽钢确认**

### 4. 透明处理
- 在项目中记录数据来源
- 标注数据抓取时间
