# 嘟嘟巴士内容生产标准

> 基于2026-03-10定时任务执行经验总结

## 1. 合规红线（必须遵守）

### 1.1 禁用词汇
**绝对禁止**出现以下词汇：
- 暴力、欺骗、恐惧、焦虑等负面情绪词
- 品牌、广告、营销、推广等商业化词汇
- 政治、宗教、社会敏感话题

**替代方案**：
- "色彩对比" → "色彩对比"
- "视觉反转" → "视觉反转"
- "标识展示" → "标识展示"
- "视觉强化" → "视觉强化"

### 1.2 品牌描述规范
**标准表达**：车身印有'嘟嘟巴士'四个大字的红色巴士

**禁止**：
- 简化为"巴士"、"大巴"
- 使用非品牌颜色（黄色、明黄、金黄等）
- 省略"嘟嘟巴士"字样

### 1.3 Slogan规范
**统一Slogan**：嘟嘟巴士！

**禁止**：
- "呢趟车稳咗"
- "这趟车稳了"
- 其他变体

## 2. 内容生产流程

### 2.1 竞品数据标准
**数据要求**：
- 必须是单日增量数据（昨天的新增量）
- 不是累计数据或近期统计

**计算方法**：
```
单日增量 = 今天数据 - 昨天数据
```

**必须包含**：
- 新增作品数
- 新增点赞数
- 互动率分析
- 平台对比
- 账号策略洞察

### 2.2 Prompt生成标准
**结构要求**（12秒视频）：
- 00:00-00:03【Hook】：视觉钩子
- 00:03-00:08【展开】：反差展示
- 00:08-00:12【收尾】：标识强化

**必须包含**：
1. 粤语开场白（45字内）
2. 完整分镜脚本
3. BGM建议
4. Tag建议
5. 合规检查清单

**创意要素**：
- 钩子类型：色彩对比、时间异常、视觉反转等
- 反差设计：旧→新、混乱→有序、危险→安全等
- 热点结合：真实且吸引眼球的话题

### 2.3 质量检查清单
**发布前必检**：
- [ ] 无敏感词汇
- [ ] 无广告用语
- [ ] 品牌描述正确
- [ ] Slogan统一
- [ ] 标识展示40-50%
- [ ] 红色巴士视觉突出

## 3. 差异化策略

### 3.1 vs 小江巴士
**他们的优势**：
- 账号矩阵规模大（15个账号）
- 高产能力（单日16条作品）
- 抖音平台表现好

**我们的机会**：
- 精品化内容（质量>数量）
- 红色巴士视觉符号
- 场景化内容（深中通道、通勤、亲子）
- 更高的互动率目标（≥124点赞/作品）

### 3.2 平台选择
**优先级**：
1. 抖音（互动率149.7）
2. 视频号（互动率42.8）
3. 小红书（测试阶段）

## 4. 数据追踪标准

### 4.1 每日监控指标
- 竞品单日增量（新增作品、点赞）
- 我们的发布量和互动数据
- 互动率对比
- 平台表现对比

### 4.2 分析维度
- 总体数据
- TOP账号表现
- 平台对比
- 账号矩阵策略
- 差异化机会
- 行动建议

## 5. 执行原则

### 5.1 合规优先
任何创意都必须先过合规检查，不能为了创意牺牲合规。

### 5.2 数据驱动
所有决策基于真实的单日增量数据，不是感觉或猜测。

### 5.3 差异化定位
不跟随竞品，而是找到自己的差异化优势。

### 5.4 质量>数量
宁可少发精品，不要多发劣质内容。

# 嘟嘟巴士视频Prompt - 四大合规标准

> 这是生成视频prompt时必须遵守的合规红线

---

## 一、关键词合规标准（必改）

### ❌ 禁止使用的高危词

**恐惧/焦虑类**：
- 视觉震撼、恐惧、压迫、惊悚、恐怖、压抑

**危险/暴力类**：
- 热点结合、劫持、暴力、违规、危险、死亡、灾难

**异常/不适类**：
- 颠倒、扭曲、异常、诡异、不适

### ✅ 必须替换成安全词

| 禁止词 | 安全替换词 |
|--------|-----------|
| 视觉震撼 | 震撼视觉 / 宏大视角 / 强烈对比 |
| 空间错位 | 氛围营造 / 视角转换 / 创意镜头 |
| 热点结合 | 热点结合 / 场景共鸣 / 节日主题 |
| 危险→安全 | 安心出行 / 稳妥守护 / 舒适保障 |
| 恐惧 | 惊讶 / 意外 / 反差 |
| 压迫感 | 视觉张力 / 空间对比 / 尺度反差 |

---

## 二、画面内容合规标准

### ❌ 绝对不能出现

**真实标识**：
- 真实地标logo
- 真实车站/机场/高速名称
- 真实车牌
- 真实公交系统标识

**危险场景**：
- 车祸、打滑、摔倒
- 冲突、拥挤推搡
- 颠倒画面、倒流水
- 恐怖特效

### ✅ 必须写成"原创+安全"

**场景要求**：
- 原创候车大厅
- 原创城市街景
- 原创道路
- 无logo、无真实地名、无真实标识展示

**画面内容**：
- 只拍：秩序、温暖、微笑、安心
- 避免：混乱、焦虑、危险、冲突

---

## 三、文案&字幕合规标准

### ❌ 不写（负面词汇）

- 恐慌、焦虑、崩溃
- 危险、混乱
- 强营销词："即刻购票"、"便宜又方便"

### ✅ 只写（正向词汇）

- 温暖、安心、稳妥
- 舒适、有序、便捷
- 信息展示："票价亲民，5.8元起"（非硬广）

### 文案风格

- 短片创意风格，非硬广
- 信息融入画面氛围
- 不做负面夸张，只做正向对比

---

## 四、音效&BGM合规标准

### ❌ 不用

**真实音效**：
- 真实广播
- 真实新闻
- 真实交通音效

**版权音乐**：
- 有版权歌曲名（如《南方姑娘》）
- 必须写成"温馨民谣风格"而非具体歌名

### ✅ 只写

**原创描述**：
- 原创轻音乐
- 原创环境音
- 温馨民谣风格
- 轻快的电子音乐

**音效描述**：
- 环境音（雨声、风声）
- 情绪音乐（紧张→舒缓）
- 品牌音效（"叮咚~"）

---

## 合规检查清单

生成每条prompt后，必须检查：

- [ ] 关键词：无禁止词，已替换成安全词
- [ ] 画面：无真实标识，无危险场景，只有原创+安全内容
- [ ] 文案：无负面词汇，只有正向表达
- [ ] 音效：无真实音效，无版权歌曲名，只有原创描述

## 🚫 违规红线（绝对禁止）

### 侵权类
1. **著作权侵权**：模仿知名广告分镜、复刻影视片段
2. **人格权侵权**：使用真人肖像、明星声音、姓名
3. **商标侵权**：使用他人注册商标、品牌标识

### 违规类
1. **公序良俗**：低俗、暴力、猎奇、歪曲历史
2. **虚假宣传**：虚构信息、夸大宣传、伪造数据
3. **未标识**：未标注"AI生成/AI辅助"

### 平台规则
1. **重复发布**：高度同质化内容
2. **搬运混剪**：无授权素材
3. **违反社区规范**：违反平台规则
