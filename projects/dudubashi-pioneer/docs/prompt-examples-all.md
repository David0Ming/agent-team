## 🎬 Prompt 1：司机的早餐

```
我要一个[生活气息]开头的12秒温情短片。

【0-3秒｜前提】：清晨6点，街道空旷。一个中年司机在路边摊吃早餐，手里拿着豆浆。

【3秒转场】：镜头从早餐摊摇到路边——一辆红色巴士驶来。司机放下豆浆，走向巴士。

【7秒内容】：
- 司机整理制服，登上驾驶座
- 乘客陆续上车，有学生、白领、老人
- 司机和每个乘客点头打招呼
- 车厢温暖，朝阳透过车窗洒进来

【结尾】：司机发动引擎，红色巴士驶入晨光中。字幕："每一次出发，都是新的故事"
```

## 🎬 Prompt 3：司机的约定

# 基于抽象框架的创作示例

**主题**: 深圳天气：晴转多云，23-28°C

---

## 视频提示词（12秒）

### 司机开场白（粤语风格，40字）
"哇！今日深圳天气正到爆！23度到28度，坐我哋嘟嘟巴士出去玩啦，有冷气唔使晒，舒服过喺屋企！"

### 画面分镜

**0-3秒 【Hook - 反常识】**
【特写】司机突然转头对镜头，夸张地擦汗："哇，热到爆！"
【镜头拉开】窗外阳光灿烂，但车内冷气十足

**4-8秒 【展开】**
【中景】车身印有'嘟嘟巴士'四个大字的红色巴士行驶在深圳街头
【切换】司机单手方向盘，另一只手指向窗外："你睇，晴转多云，最啱出行！"
【远景】窗外深圳城市景色，天空从晴朗渐变到云层

**9-12秒 【收尾 - 互动】**
【特写】司机竖大拇指，眨眼："23到28度，坐巴士最舒服！你哋话系唔系？"
【定格】车身"嘟嘟巴士"大字特写，司机笑容

### 技术参数
- 画幅：9:16 竖屏
- 色调：明亮暖色，阳光感
- 节奏：前快后慢，0-3秒快切，4-12秒平稳
- 镜头语言：特写→中景→远景→特写

### BGM 建议
轻快电子音乐，0:00-0:03 节奏感强，0:04-0:12 舒缓

### 建议 Tag
#嘟嘟巴士 #深圳天气 #出行舒适 #大湾区出行 #城际巴士

---

## 创作亮点

1. **Hook 设计**：用"反常识"——外面热，车里凉，制造反差
2. **粤语地道**：用"正到爆"、"最啱"、"你哋话系唔系"等地道表达
3. **视觉强化**：3次展示"嘟嘟巴士"（车身大字、司机提及、结尾特写）
4. **情感连接**：用天气这个共同话题，引发共鸣
5. **行动召唤**：结尾互动式提问，引导评论

---

- 新版：基于框架自由创作，粤语地道，有创意（灵活）
