# Seedance（即梦）视频提示词 Skill 学习

> 来源: https://github.com/songguoxs/seedance-prompt-skill

## 核心能力

| 能力 | 说明 |
|------|------|
| 纯文本生成 | 无需参考素材，纯文字生成 |
| 一致性控制 | 上传参考图片保持角色/场景统一 |
| 运镜复刻 | 上传参考视频精准复刻镜头语言 |
| 创意复刻 | 模仿创意转场、广告、电影片段 |
| 剧情创作 | 根据分镜图片自动生成剧情 |
| 视频延长 | 已有视频前向/后向延长 |
| 声音控制 | 音色参考、对白生成、音效设计 |
| 一镜到底 | 连贯长镜头平滑过渡 |
| 视频编辑 | 角色替换、剧情颠覆 |
| 音乐卡点 | 画面与音乐节拍匹配 |

## 工作流（四步）

```
1. 描述创意 → "一段古风武侠打斗"
2. 确认参数 → Claude 询问时长、比例、参考素材
3. 生成提示词 → ≤15秒生成2-3个版本，>15秒多段拼接
4. 微调优化 → 调整镜头、语言、音效
```

## 关键技巧

### 1. 时间戳分镜法
按秒精确控制每个镜头：
```
15秒仙侠高燃战斗镜头，金红暖色调
0-3秒：低角度特写主角蓝袍衣摆...
4-8秒：环绕摇镜快切...
9-12秒：仰拍拉远定格慢放...
13-15秒：缓推特写主角落地收剑...
```

### 2. 技术参数
在提示词开头声明：
- 画幅比（16:9、9:16）
- 帧率
- 色调（冷色调、暖色调）
- 镜头风格

### 3. 禁止项声明
```
禁止出现水印、字幕、Logo
```

### 4. @引用语法
- @图片1~9 — 角色参考、场景背景
- @视频1~3 — 运镜、动作参考
- @音频1~3 — 背景音乐、音效

### 5. 台词格式
```
台词1（角色，情绪）：台词内容
台词2（角色，情绪）：台词内容
```

## 整合到嘟嘟巴士

**重要要求**：视频时长 **≤12秒**

结合司机人设的视频提示词模板：

```
12秒巴士司机视角竖屏视频，金红暖色调
0-3秒：【Hook】司机转头对镜头吐槽，表情夸张
4-8秒：【展开】司机单手方向盘，窗外景色闪过
9-12秒：【反转/收尾】司机眨眼竖大拇指，"这趟车稳了！"

BGM：《大风吹》0:08-0:20（卡点）
禁止出现其他巴士logo、水印
```

## 应用建议

- 嘟嘟巴士视频用 9:16 竖屏格式
- **时长严格控制在 12 秒以内**
- 每3-4秒一个镜头切换
- 融入司机人设风格
- 热点话题+司机视角
