# AI文生视频提示词创作指南

> 来源: 飞书wiki - 德里克文
> 原文: https://zhuanlan.zhihu.com/p/711848787

## 一、结构化

当提示词具备清晰的结构时，生成效果最为显著。

### 简单公式
```
[镜头动态] + [主体描述] + [动作描述]
```

### 复杂公式
```
[镜头动态] + [光影描述] + [主体描述] + [主体运动] + [环境描述] + [主体细节描述] + [其他描述：情绪氛围/美学风格]
```

## 二、示例

### 无结构提示词（效果差）
> "一只狗在跳。"

### 有结构提示词（效果好）
> 镜头跟随（镜头动态）一只活泼的小狗在草地上（主体描述），它欢快地跳跃着（情绪氛围+主体动作）。小狗的毛发在阳光下闪闪发光，显得格外精神（主体细节描述），背景是一片开阔的草地，微风轻轻吹过，草丛随之摇曳（环境描述），真实照片风格（美学风格）。

## 三、适用工具

- 即梦
- 可灵
- 清影
- Veo 3

## 四、结合司机人设

在生成嘟嘟巴士视频提示词时，应该：
1. 先确定热点主题
2. 用结构化公式生成英文画面描述
3. 融入司机第一人称口语风格
4. 前3秒必须有冲突或笑点
