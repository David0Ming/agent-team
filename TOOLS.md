## OpenClaw 常用命令

```bash
# 状态查看
openclaw status              # 查看系统状态
openclaw logs                # 查看日志
openclaw logs --follow       # 实时日志

# Gateway 管理
openclaw gateway restart     # 重启Gateway
openclaw gateway status      # Gateway状态

# 配置管理
openclaw config get <path>   # 获取配置
openclaw config set <path> <value>  # 设置配置
```
