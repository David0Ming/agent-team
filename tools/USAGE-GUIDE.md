# 工具使用指南

## 快速开始
```bash
bash ~/.openclaw/workspace/tools/quick-start.sh
```

## 常用工具

### 健康检查
```bash
bash ~/.openclaw/workspace/tools/agent-health-check.sh
python3 ~/.openclaw/workspace/tools/agent-test.py
```

### 状态查看
```bash
bash ~/.openclaw/workspace/tools/status-summary.sh
bash ~/.openclaw/workspace/tools/version-info.sh
```

### 维护工具
```bash
bash ~/.openclaw/workspace/tools/cleanup.sh
bash ~/.openclaw/workspace/tools/backup-config.sh
```
